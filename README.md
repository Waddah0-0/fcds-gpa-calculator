# GPA Calculator

A clean, precise GPA calculator web application built with HTML, CSS, and JavaScript.

## Features
- Calculates Semester GPA and Cumulative GPA.
- Custom grading scale points mapping.
- Fully responsive dark-themed UI.
- Local calculation only (no server required).

## Usage
Simply open `index.html` in any modern web browser, or visit the live application at:
**[https://waddah0-0.github.io/fcds-gpa-calculator/](https://waddah0-0.github.io/fcds-gpa-calculator/)**
